BrianRepro

Distributed Microservices Chat Application

About This Demo

This is a reproduction environment demonstrating distributed microservices communication patterns using Spring Boot, Apache Artemis message queues, and real-time WebSocket connections.

The system showcases inter-service communication, message queuing, and containerized deployment with comprehensive logging and monitoring.

Technology Stack

Backend: Spring Boot 2.7.18 with WebSocket support

Message Queue: Apache Artemis for JMS-based communication

Reverse Proxy: Caddy for load balancing and routing

Orchestration: Docker Swarm with custom "repro" services

Logging: Graylog integration with GELF protocol

Service Architecture

reproartemis (Message Queue)
    ↓
reproping ← → repropong (Chat Services)
    ↓
reprocaddy (Reverse Proxy)

Available Services

Ping Service Pong Service
